# pi-learnings

Generic evidence-backed learning capability for Pi extensions.

`pi-learnings` owns reusable learning contracts and the canonical `learning` action tool. During migration it re-exports the existing `spark-learnings` implementation so local plural `.learnings/` stores remain local-only and do not move.

Learnings stay distinct from recall/memory: they are evidence-backed records with lifecycle state, not automatic conversational memory.

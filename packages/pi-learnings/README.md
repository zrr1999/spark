# pi-learnings

Generic evidence-backed learning capability for Pi extensions.

`pi-learnings` owns reusable learning contracts, local/user learning stores, lifecycle operations, Markdown export/import, and the canonical `learning` action tool. Repo/workspace records stay in plural `.learnings/`; user records stay under the configured user learning directory. These stores remain local-only unless explicitly exported for review/sharing.

Learnings stay distinct from recall/memory: they are evidence-backed records with lifecycle state, not automatic conversational memory.

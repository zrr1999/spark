# spark-learnings

Compatibility facade plus reflection pipelines.

- **LearningStore / `learning` tool** — owned by [`@zendev-lab/spark-memory`](../spark-memory) (`./learning`). Prefer importing from there.
- **Reflection** (`reflection-*`) — still shipped from this package until the next memory consolidation slice.

Repo/workspace records stay in plural `.learnings/`; user records stay under the configured user learning directory.
